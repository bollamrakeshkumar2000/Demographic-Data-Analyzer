
# Demographic Data Analyzer

FreeCodeCamp Data Analysis with Python - Project 2

## Project Description
Analysis of demographic data from the 1994 Census database using Pandas to uncover patterns and insights about income levels.

## Questions Answered
1. Race distribution in the dataset
2. Average age of men
3. Percentage with Bachelor's degrees
4. Income patterns by education level
5. Work hours analysis
6. Geographic income patterns
7. Occupation trends in India

## Technologies Used
- Python
- Pandas
- NumPy
